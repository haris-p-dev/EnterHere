## Project In Progress

**EnterHere** is a small **Java Full Stack project** demonstrating basic data management using objects and generics. This project is great for practicing OOP concepts, collections, and simple backend logic.

## Features

- Uses **Java Generics** for storing objects with unique IDs.
- Organized class structure with packages for clean and maintainable code.
- Examples of basic object operations (CRUD-like behavior).
- Full stack ready structure for future expansion.

## Technologies

- **Java 17**
- Collections API (ArrayList, Map, Set)
- IntelliJ IDEA
